# 🎮 Colin Ackroyd - Game Developer Portfolio

Welcome to the source code of my personal portfolio website!

This site is a showcase of my journey in game development, featuring details and technical highlights of my main projects. It is built with ⚛️ **React**, **TypeScript**, and styled using **Tailwind CSS** with a unique comic-book aesthetic.

## Featured Projects
- **R-Type:** A complete multiplayer recreation of the classic arcade shooter built from scratch in C++. Features a custom ECS, custom networking protocol using Asio, and SFML rendering.
- **Zappy:** A multiplayer network game with a server in C, autonomous AI clients, and a graphical client in C++ with SFML.
- **2D Platformer:** A fast-paced Unity Christmas platformer developed in 48 hours for a Game Jam.
- **Horror Prototype:** A 3D horror game prototype built in Unity, focusing on atmospheric lighting and tense gameplay mechanics.

## Tech Stack
- **Framework:** React (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Comic-style UI)
- **Animations:** Framer Motion
- **Icons:** Lucide React

## How to Run Locally

If you'd like to test or view this site on your own machine:

1. Clone the repository
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
